# VideoJuego

Juego estilo Galaga / Space Invaders hecho con HTML, CSS y JavaScript.

## Descripción
Este proyecto es un videojuego arcade en el que controlas una nave espacial, disparas a los enemigos y tratas de sobrevivir el mayor tiempo posible.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Canvas API

## Cómo ejecutar
1. Descarga o clona el repositorio.
2. Abre `index.html` en tu navegador.

## Controles
- `←` mover a la izquierda
- `→` mover a la derecha
- `Espacio` disparar

## Archivos principales
- `index.html`: estructura del juego
- `style.css`: estilos e interfaz
- `game.js`: lógica del juego

## Características
- Nave controlable
- Disparos
- Enemigos en movimiento
- Sistema de vidas
- Puntuación
- Pantalla de game over
